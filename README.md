## ▶️ How To Run Any Project
```bash
# 1. Clone the repo
git clone https://github.com/SteveTawali/automation-portfolio.git

# 2. Go into any project
cd project_1_login_suite

# 3. Install dependencies
pip install -r requirements.txt
playwright install

# 4. Run tests
pytest tests/ -v --headed
```
